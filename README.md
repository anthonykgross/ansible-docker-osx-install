# ansible-docker-osx

```bash
brew install python
pip install ansible
```

```bash
git clone https://github.com/anthonykgross/ansible-docker-osx.git
cd ansible-docker-osx
```

```bash
ansible-playbook install.yml 
```

```bash
ansible-playbook uninstall.yml 
```
